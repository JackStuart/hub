<Card title="Applied Skills" icon="star"> 
Applied skills are a new type of credential that tests your hands on experience with the products. It will only practical tasks you complete with Microsoft products. They are free at the moment so a great way to practice for free but that can change soon.
</Card>

<Card title="Microsoft GitHub" icon="star"> 
While doing GitHub labs is not free, like those on Microsoft Learn, they are complete lab exercises that guide you through projects and help you get familiar with the products you will be tested on. For Azure, at the end of all labs there is self destruct PowerShell script, always run it when you finish to avoid additional costs.
</Card>

<Card title="Microsoft Learn" icon="star"> 
Microsoft Learn sandboxes are mini-labs that are part of Microsoft Learn modules. Easy and simple way to practice using the products you will get tested on.
</Card>

<Card title="Guided Labs" icon="star"> 
These are basically interactive videos, they are free and easy to navigate but the issue is that it is not a live environment, always good to supplement with real labbing with Microsoft Products.
</Card>
